🚀 快速開始
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1️⃣  安裝依賴
   $ ./setup.sh
   或
   $ pip install -r requirements.txt

2️⃣  準備數據（可選）
   $ python scripts/download_wiki.py

3️⃣  建立索引
   $ python build_index.py

4️⃣  查詢 IDF 值
   $ python query_idf.py 蝙蝠俠
   $ python query_idf.py 台灣
   $ python query_idf.py 人工智慧
